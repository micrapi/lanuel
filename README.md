# Larnuxele
A Laravel - Nuxt 3 - Element Plus starter kit.

## Installation

### Clone this repository
```
git clone https://github.com/micrapi/larnuxele.git

cd larnuxele
```

### Laravel
```bash
composer install

cp .env.example .env

php artisan key:generate --ansi
```

### NuxtJs
```bash
yarn install

yarn build
```

### PM2
```bash
npm install -g pm2

pm2 start
```
